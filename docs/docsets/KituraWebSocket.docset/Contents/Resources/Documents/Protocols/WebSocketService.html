WebSocketService

public protocol WebSocketService: class

The WebSocketService protocol is implemented by classes that which to be WebSocket server side end points. An instance of the WebSocketService protocol is the server side of a WebSocket connection. There can be many WebSocket connections connected to a single WebSocketService protocol instance. The protocol is a set of callbacks that are invokes when various events occur.
